# Syntax for the Just programing language

## Functions
```
function name(params)
    action
end
```

## Loops
```
for var in iterable
    action
end
```
```
while condition
    action
end
```

## control flow
```
if condition
    action
else if condition
    action
else
    action
end
```

## variable assignment
```
int num = 5
string name = "John"
float price = 10.99
```

## operators

add: +
substract: -
